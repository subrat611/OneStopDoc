# Introduction

- SOLID principles were introduced by Robert C. Martin (Uncle Bob).
- SOLID principles were formulated to provide a framework for software developers to design and build software systems that are easy to maintain, extend, and understand.
- SOLID stands for

  - `Single Responsibility Principle (SRP)`

  - `Open/Closed Principle (OCP)`

  - `Liskov Substitution Principle (LSP)`

  - `Interface Segregation Principle (ISP)`

  - `Dependency Inversion Principle (DIP)`
