# JavaScript

- JavaScript, born in 1995 as "Mocha" and later "LiveScript," evolved into a dynamic web language.
- It gained prominence with the DOM in 1997, facilitating web interactivity.
- The Browser Wars prompted ECMAScript standardization in the late '90s, cementing JavaScript's role.
- The early 2000s introduced AJAX, driving Web 2.0. jQuery simplified DOM manipulation in 2006.
- Node.js revolutionized server-side JavaScript in 2009. ECMAScript 6 (2015) brought language enhancements.
- Modern frameworks like Angular, React, and Vue.js emerged in the 2010s, while WebAssembly enabled high-performance code.
